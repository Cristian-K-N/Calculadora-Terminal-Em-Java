# Calculadora-Terminal-Em-Java

Calculadora simples em Java que roda direto pelo terminal, suportando as quatro operações básicas.

Você vai precisar do JDK 8 ou superior, e abrir o terminal como: CMD,PowerShell,Bash e entre outros...

Encontre o caminho do arquivo "\calculate\src" e digite no terminal 
cd (o caminho do arquivo).

Logo após compile o código com o comando:
javac Calculate.java

Execute o comando passando os argumentos:
java Calculate somar 5 5,
java Calculate subtrair 5 5,
java Calculate multiplicar 5 5,
java Calculate dividir 5 5.
